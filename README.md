# Agenda 1. リサーチ結果 — 「なぜ空白か」と「事業は成立するか」

## 1-1. 空白の正体：「供給の罠」であって「需要の罠」ではない

佐々木氏が前回「競合がいないのはなんでなんだろう」と問うた。答えは**需要側ではなく供給側**にある。

### 需要は存在し、成長している（＝空白の理由ではない）

| 指標 | 数値 | 出典 |
|------|------|------|
| 訪日ベトナム人数（2025年） | **67.9万人**（過去最高、前年比+9.2%） | JNTO訪日外客統計 |
| 旅行消費額（2025年） | **2,055億円**（過去最高） | 観光庁インバウンド消費動向調査 |
| 1人当たり消費額（2025年4-6月） | **30.1万円**（全市場平均+6.2万円） | 観光庁・訪日ラボ |
| FIT・高単価シフト | 「プライベート感があり、アレンジの利く高単価ツアーを求めるようになってきている」 | JNTOハノイ事務所（2023年12月） |

### 供給が構造的に枯渇している（＝空白の主因）

| 要因 | 内容 | 評価 |
|------|------|------|
| **(A) 制度的要因** | ベトナム語は通訳案内士の10言語対象外。ただし2018年法改正で無資格有償ガイドは合法 | **実は追い風**（参入障壁なし） |
| **(B) 人材プール** | 在日ベトナム人60万人だが技能実習20.4万人が中心。日本語堪能な高度人材（技人国4.5万人）はIT・通訳・貿易を選好 | **最大のボトルネック** |
| **(C) 経済合理性** | ガイド日当2〜3万円＋季節変動大 ＜ 安定した通訳・専門職の月給 | ガイドが職業として選ばれにくい |

### ロジックツリー：なぜ空白なのか

```
競合不在（ベトナム語×富裕層×プライベート×京都）
├── 需要側：存在する（＝空白の理由ではない）
│    ├─ 訪日ベトナム人 67.9万人(2025)、消費額2,055億円（過去最高）
│    ├─ 1人当たり消費額30.1万円＝全市場平均を6.2万円超過
│    └─ JNTOが「高単価プライベートを求めるようになった」と報告
│
├── 供給側：枯渇している（＝空白の主因）
│    ├─ 制度：ベトナム語は通訳案内士対象外（でも無資格解禁で参入自由）
│    ├─ 人材：在日ベトナム人60万人だが技能実習中心、観光に回らない
│    └─ 経済合理性：ガイド日当2-3万円＋季節変動 ＜ 安定職
│
├── 事業モデル側：マッチングPFが規模の経済で失敗
│    └─ 言語特化の失敗ではなく汎用PFのスケール不成立
│
└── VISA障壁：FIT化の最大の摩擦
     └─ タイ（ビザ免除・リピーター70%）では成立、ベトナムでは未成立の一因
```

**佐々木氏への読み替え：** 「みんな必死に考えてるのになんでここにたどり着けてないんだ？」への答えは、**ベトナム語ガイドを組成する方法を誰も持っていなかった**ということ。需要がないから空白なのではなく、供給を作れなかったから空白。逆に言えば、供給を作れる事業者にとっては参入可能。
